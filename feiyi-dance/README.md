# 非遗舞鉴 - 少数民族舞蹈数字保护平台

这是一个展示性的网站原型，用于"非遗舞鉴"项目的创意阶段展示。

## 功能特点

- 响应式设计，适配各种设备
- 简洁美观的界面设计
- 包含首页和关于页面
- 联系表单功能
- 基本的交互效果

## 使用技术

- HTML5
- CSS3
- JavaScript
- GitHub Pages (托管)
- Netlify Forms (表单处理)

## 如何运行

1. 克隆或下载本仓库
2. 直接打开`index.html`文件即可在浏览器中查看

## 部署说明

1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 连接Netlify账户以启用表单功能
